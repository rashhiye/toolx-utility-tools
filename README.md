**ToolX** is a comprehensive platform offering **25+ free online tools** for productivity, file conversion, text processing, and utility tasks. Built with React, TypeScript, and Tailwind CSS for maximum performance and user experience.

## 🌟 Features

- **25+ Free Online Tools** - PDF, Image, File, Generator, Text, and Utility tools
- **No Signup Required** - Use all tools instantly without registration
- **Privacy-First** - All processing happens locally in your browser
- **Responsive Design** - Beautiful glass-morphism UI works on all devices
- **SEO Optimized** - Professional content and metadata for search engines
- **AdSense Ready** - Includes privacy policy, terms, and proper legal pages
- **Fast & Lightweight** - Built with Vite for instant page loads
- **Fully Typed** - Written in TypeScript for reliability

## 🚀 Getting Started

### Prerequisites
- **Node.js** 18+ or **Bun** runtime
- **npm**, **yarn**, **pnpm**, or **bun** package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/rashhiye/toolx.git
cd toolx

# Install dependencies
bun install
# or: npm install

# Start development server
bun run dev
# or: npm run dev
```

### Build for Production

```bash
bun run build
# or: npm run build

# Preview production build
bun run preview
# or: npm run preview
```
